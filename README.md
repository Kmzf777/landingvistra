# Vistra Landing Pages

Projeto Next.js com App Router, Tailwind CSS e TypeScript.

## Estrutura

- `/app`: Código fonte da aplicação (App Router)
  - `/app/dashboard`: Painel administrativo
- `/public`: Arquivos estáticos

## Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## Hospedagem na Vercel

Este projeto está pronto para ser hospedado na Vercel. Basta conectar seu repositório Git e fazer o deploy.
